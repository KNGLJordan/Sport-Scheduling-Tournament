### setup_cdmo_env.sh

Bash file useful to setup a python virtual environment with the necessary requirements.

<b>Make the script executable</b>

```bash
chmod +x setup_cdmo_env.sh
```

<b>Run the script</b>

```bash
./setup_cdmo_env.sh
```
